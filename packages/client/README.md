# @aleph/client

The agent-facing Aleph SDK — THE target. Resolve, rank by computed trust, invoke, pay via escrow, verify receipts, and compose across nodes.

Part of the [Aleph Protocol](https://github.com/Leonardo-Corte/aleph-protocol) — a thin-waist protocol for an agent-native web: how machines **find, trust, act, pay, and prove**.

## Install

```bash
npm install @aleph/client
```
