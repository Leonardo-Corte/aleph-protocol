# @aleph/node

The Aleph node runtime: a capability provider that publishes a Manifest, verifies grants, validates input schema, settles payment, and returns signed receipts.

Part of the [Aleph Protocol](https://github.com/Leonardo-Corte/aleph-protocol) — a thin-waist protocol for an agent-native web: how machines **find, trust, act, pay, and prove**.

## Install

```bash
npm install @aleph/node
```
