# @aleph/registry

The Aleph registry: the discovery multiplier. Nodes register; agents RESOLVE by capability; registries federate via gossip.

Part of the [Aleph Protocol](https://github.com/Leonardo-Corte/aleph-protocol) — a thin-waist protocol for an agent-native web: how machines **find, trust, act, pay, and prove**.

## Install

```bash
npm install @aleph/registry
```
