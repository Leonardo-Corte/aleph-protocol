# @aleph/core

The thin waist: identity (did:key/web), signed envelopes, bounded grants, manifests, schema validation, the settlement interface, and trust (attestations + receipt chains). No servers — portable to any runtime.

Part of the [Aleph Protocol](https://github.com/Leonardo-Corte/aleph-protocol) — a thin-waist protocol for an agent-native web: how machines **find, trust, act, pay, and prove**.

## Install

```bash
npm install @aleph/core
```
