# @aleph/mcp

Aleph as an MCP server. Exposes aleph_resolve (FIND) and aleph_invoke (ACT + PROVE) so any MCP-capable agent uses Aleph natively.

Part of the [Aleph Protocol](https://github.com/Leonardo-Corte/aleph-protocol) — a thin-waist protocol for an agent-native web: how machines **find, trust, act, pay, and prove**.

## Install

```bash
npm install @aleph/mcp
```
